# 🔐 Password Generator
A responsive Password Generator web application built using React and Tailwind CSS, featuring customizable password options, password strength indication, clipboard copy functionality, and password history tracking.

---

## ✨ Features

- 🔑 Generate secure random passwords
- 📏 Adjustable password length
- 🔠 Include uppercase letters
- 🔡 Include lowercase letters
- 🔢 Include numbers
- 🔣 Include special characters
- 📋 One-click copy to clipboard
- 💪 Password strength indicator
- 🕒 Recent password history
- 📱 Fully responsive design
- 🎨 Clean and modern user interface

---

## 🛠️ Tech Stack

- React.js
- Tailwind CSS 
- Vite
- JavaScript
